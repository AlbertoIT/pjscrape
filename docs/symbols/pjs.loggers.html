Namespace pjs.loggers

Defined in:
pjscrape.js

Logger namespace. You can add new loggers here; new logger classes should probably extend pjs.loggers.base and redefine the log method. 


        // create a new logger
        pjs.loggers.myLogger = function() {
            return new pjs.loggers.base(function(msg) { 
                // do some special logging stuff
            });
        };
        // tell pjscrape to use your logger
        pjs.config({
            log: 'myLogger'
        });
